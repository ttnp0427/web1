HTML

생활코딩을 이용해서 자신만의 웹만들기

html이란 Hypertext Markup Language로 웹 페이지와 웹 응용프로그램을 만들기 위한 표준 마크업 언어이다. 웹 브라우저는 웹 서버나 로컬 스토리지에서 HTML 문서를 수신하여 멀티미디어 웹 페이지로 렌더링한다. HTML은 웹페이지의 구조를 의미론적으로 설명하고 있으며, 문서의 외관을 위해 원래 포함된 단서들을 포함한다.
HTML 요소는 HTML 페이지의 구성 요소이다. HTML 구성으로 이미지와 대화형 양식과 같은 다른 오브젝트가 렌더링된 페이지에 포함될 수 있다. 표제, 단락, 목록, 링크, 인용문 및 기타 항목과 같은 텍스트에 대한 구조적 의미를 나타냄으로써 구조화된 문서를 작성할 수 있는 수단을 제공한다. HTML 요소는 각괄호를 사용하여 작성된 태그에 의해 기술된다.
